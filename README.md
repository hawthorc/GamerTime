<h1>2D Multiplayer Game</h1>

<h2>Description</h2>
This project is a top down, 2D cooperative multiplayer video game, created in Java without using a game engine. All game and networking logic, features, and assets are created with existing Java libraries, resulting in a fun and challenging learning experience!
<br />


<h2>Languages and Utilities Used</h2>

- <b><a href="https://www.java.com/en/">Java</a>, via <a href="https://eclipseide.org/">Eclipse IDE</a></b>
- <b><a href="https://www.piskelapp.com/">Piskel</a>, an online pixel editing tool</b>
